Simple game of snake created using python and its built-in tkinter module. Collect orange squares and avoid hitting your tail. You can move through an edge of the area to continue from the opposite side.

START button to start a game. Changes to STOP when a game is running.

WASD or arrow keys to control snake.

SPEED button toggles between refresh rates for the game. Default is 5. Speed setting are: 1: 300ms, 2: 250ms, 3: 200ms, 4: 150ms, 5: 100ms, 6: 50ms, 7: 30ms
